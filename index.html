Survol des techniques et technologies nécessaires à sa réalisation

C’est le langage de balisage HTML et le CSS que nous utiliserons pour réaliser ce travail. Ayant comme objectif de créer un site Web de plusieurs pages, de nombreuses techniques seront à apprendre. Tout d’abord, nous devrons apprendre comment créer un menu et lier les différentes pages entre elles, en plus d’insérer un pied de page soit qui sera présent partout. Puisque notre site Web rassemble les œuvres d’une aquarelliste et donc une quantité importante d’images, nous devrons développer nos aptitudes en CSS afin de bien disposer les images, notamment en mode damier en prenant en compte qu’elles ne sont pas toutes de la même taille. Afin de rendre l’expérience de navigation intéressante, nous aimerions découvrir une option qui permet d’afficher les images dans un format plus grand lorsqu’on clique dessus. Pour nous aider à réaliser ce travail, nous consulterons le site Web de la W3Schools et les guides offerts par Mozilla. À titre d’inspiration, nous avons regardé nombreux site Web d’artistes et nous nous inspirerons du site Web de Jacynthe Carrier. La création de ce site Web nous permettra de découvrir et d’explorer de nouvelles fonctions HTML et CSS et de nous assurer que toute l’équipe maîtrisera par la suite ces langages de programmation.
